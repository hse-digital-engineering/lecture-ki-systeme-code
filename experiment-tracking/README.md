# Experiment tracking

This folder contains a experiment tracking example as a [notebook](experiment-tracking-example.ipynb).

The experiment tracking is done with mlfow.

Before you start

```bash
# init the project 
uv sync

# start mlflow
uv run mlfow server
```

Open the mlflow ui [http://127.0.0.1:5000](http://127.0.0.1:5000)
